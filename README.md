🏙️ Buenos Aires Apartment Price Predictor

This project leverages machine learning to predict apartment prices in Buenos Aires, focusing on properties valued under $400,000. Moving beyond traditional descriptive analysis, it applies predictive data science techniques to uncover patterns and forecast real estate prices with accuracy.

Building on prior experience analyzing the housing market in Mexico, this project marks a transition from data wrangling and visualization to model-driven insights. It involves cleaning and preparing real-world housing data, exploring key features that influence pricing, and training machine learning models to generate reliable predictions.

The final model provides a practical tool for understanding how factors like location, size, and property characteristics impact apartment prices, offering value to potential buyers, investors, and real estate professionals.

🚀 Key Highlights
  -**Applied end-to-end data science workflow** (wrangling → exploration → modeling)
  -**Built and evaluated machine learning regression models**
  -**Focused on real-world affordability constraints** (under $400K market segment)
  -**Transformed raw data into actionable pricing insights**
